# Robotic Vision and Path Following - A Review and Implementation

This repository contains the simulation files and the theoretical analysis from my Diploma Thesis at the University of Western Macedonia.

## Overview
The project focuses on the integration of machine vision with robotic control systems. It provides an extensive literature review of autonomous navigation and implements specific path-following scenarios in a simulated environment.

## Technical Implementation
The practical simulations (detailed in Chapter 5 of the thesis) were developed using:
* **MATLAB**
* **Robotics Toolbox for MATLAB** by Peter Corke

The simulations evaluate how vision-based control laws can be applied to maintain a robot's trajectory and ensure accurate path following.
